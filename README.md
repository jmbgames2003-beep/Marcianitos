# 👾 Marcianitos

Juego arcade estilo Space Invaders hecho con HTML5 Canvas puro. Instalable como app en móvil y escritorio (PWA).

**[▶ JUGAR ONLINE](https://TU_USUARIO.github.io/marcianitos)**

---

## 🎮 Controles

| Plataforma | Mover | Disparar | Pausa |
|---|---|---|---|
| **Teclado** | ← → | Espacio (mantener) | P / Esc |
| **Móvil** | Botones ◀ ▶ | Botón 🔴 (mantener) | ⏸ |

## ✨ Características

- Múltiples niveles con dificultad progresiva
- Invasores más rápidos y más filas a cada nivel
- OVNI con puntuación aleatoria (50–300 pts)
- Sonido retro generado por Web Audio API
- Efectos CRT (scanlines + viñeta)
- **Instalable como app** en Android, iOS y escritorio (PWA)
- Funciona offline tras la primera visita

## 📱 Instalar como app

### Android (Chrome)
1. Abre el enlace en Chrome
2. Toca el menú ⋮ → **"Añadir a pantalla de inicio"**

### iPhone / iPad (Safari)
1. Abre el enlace en Safari
2. Toca el botón compartir → **"En la pantalla de inicio"**

### Escritorio (Chrome / Edge)
1. Abre el enlace
2. Haz clic en el icono 📥 en la barra de direcciones

## 🚀 Despliegue en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Source* selecciona **Deploy from a branch → main → / (root)**
4. En unos segundos estará en `https://TU_USUARIO.github.io/marcianitos`

## 🛠 Tecnologías

- HTML5 Canvas (sin librerías)
- Web Audio API (sonido procedural)
- Service Worker (PWA / offline)
- CSS puro con efectos CRT
